## Usage:

#### Display your tasks:

```
t
```

#### Add some tasks:

```
t this is a task 
t "another task" 
```

#### Mark them as completed:

```
t 1-2
t 1
```

#### Delete a task:

```
t d 1
```

#### Clear your completed tasks:

```
t cls
```

#### View your old tasks:

```
t old
```

#### List commands:

```
t help
```
